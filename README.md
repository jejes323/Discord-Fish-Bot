# 낚시 봇 시스템 🎣
디스코드에서 낚시 게임을 즐기세요 !

물고기르 잡고 서버내 낚시왕이 되어보세요.

## 사용 기술 스택 📚
- 런타임 환경
  - Node.js v22.18.0
- 핵심 프레임워크 & 라이브러리
  - discord.js (v14.18.0)
  - @discordjs/builders (v1.10.1)
- 데이터베이스
  - better-sqlite3 (v12.4.6)

## 프로젝트 구조 🧩

Discord-Fish-Bot/

├── fish.js              # 메인 봇 로직

├── fish_database.db     # SQLite 데이터베이스

└── package.json         # 프로젝트 의존성 관리

## 주요 기능 

- 🎣 낚시 시스템 (쿨다운 포함)
- 📦 인벤토리 시스템 (희귀도별 필터링)
- 💰 물고기 거래 시스템
- 🎲 확률 기반 물고기 획득
- 💌 DM 알림 기능
- 🔧 관리자 명령어 (물고기 추가/관리)

## 추후 업데이트 예정
- 상점 낚싯대 추가
- 낚싯대 물고기 등급 확률 조정
- 물고기 길이 추가
- 랭킹 시스템 도입
- 도감 도입

+ /물고기추가 [등급] [이름] [가격] - 물고기 DB에 등급, 이름, 가격에 맞게 추가
+ /물고기삭제 [이름] - 물고기 DB에서 물고기 삭제
