# Discord-Fish-Bot

## 낚시 봇 시스템
낚시를 통해 여러 물고기를 낚고 상점에 판매하여 수익을 내는 구조

수익금으로 상점에서 새로운 낚시대를 구입 가능

가장 보유금이 많은 플레이어가 랭커 등록



이외 도감작을 통해 소소한 재미를 볼 수 있음

## 업데이트 내용

2025.03.31
+ /버튼 - 버튼 패널 생성

2025.11.24
+ 버튼 일부 수정
+ 낚시하기
  + 낚시하기 버튼 클릭 시 1분 동안 낚시를 진행, 진행 중인 경우 낚시 불가능
  + 확률을 통해서 높은 등급의 물고기 낚기
  + 1분 뒤 성공 메시지를 개인 DM으로 전송, 성공 메시지는 물고기 등급, 이름, 가격을 포함
+ /물고기추가 [등급] [이름] [가격] - 물고기 DB에 등급, 이름, 가격에 맞게 추가
+ /물고기삭제 [이름] - 물고기 DB에서 물고기 삭제
